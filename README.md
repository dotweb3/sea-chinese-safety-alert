# 东南亚与澳新中文安全预警 Web MVP

这是一个面向公众的中文安全预警原型，覆盖：

- 东南亚：高薪招聘诱骗、跨境失联、园区强迫犯罪。
- 澳大利亚/新西兰：假冒权威、虚拟绑架、投资/加密诈骗、AI 深伪社工。
- 数据方式：公开来源 + 人工录入 + 审核后发布。

## 打开方式

直接用浏览器打开 `index.html` 即可。也可以在目录中启动一个静态服务：

```bash
python3 -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。

## 已实现的 MVP 功能

- 总览看板：今日重点、指标、最新套路、快速求助。
- 风险地图：7 个重点地区，按风险等级筛选。
- 套路库：30 条公开来源支撑的预警/案例化条目，可搜索和按类型筛选。
- 求助清单：12308、泰国、缅甸、澳大利亚、新西兰和金融止损入口。
- 订阅偏好：按地区、人群、频率保存到本机浏览器。
- 报料入口：默认进入审核池，自动提示可能的个人敏感信息。
- 编辑台：人工录入预警、来源归档、审核池、周报草稿。

## 内容安全边界

- 不公开个人隐私。
- 不发布未核实指控。
- 不复述可被犯罪者复用的详细话术和操作步骤。
- 高风险内容只给识别信号、避险动作和官方求助渠道。

## 后续接入建议

- 后端：把 `alert`、`source`、`region`、`modus_operandi`、`help_contact`、`case_summary`、`risk_score` 做成数据库表。

- 舆情：接入 RSS、政府公告、警方通报、主流媒体和公开社媒链接。
- App：复用当前信息架构，增加推送、离线求助清单、家人安全卡和旅行前风险检查。
